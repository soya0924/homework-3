# Graphic-Line-Wave

一個基於 HTML Canvas 的互動式波浪動畫專案。

## 功能特點
* 純 JavaScript 實現
* HTML Canvas 動畫效果
* 繪製波浪圖案
* Arduino 可調電阻控制
* 即時波浪參數調整
* 動態顏色變化

## 安裝說明
1. 複製專案到本地
2. 開啟 index.html
3. 連接 Arduino 裝置（選配）

## 使用方法
1. 直接開啟網頁觀看波浪動畫
2. 點擊「連接 Arduino」按鈕來使用可調電阻控制
3. 轉動可調電阻來改變波浪的：
   - 振幅
   - 速度
   - 顏色
   - 線條粗細
